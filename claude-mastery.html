Day 1: Meet Claude Properly

LEARN

Learning Objective

Understand Claude's architecture, character, and how it differs from other LLMs. Learn to write effective system prompts that shape Claude's behavior for specific use cases.

Learning Blocks ~3 hours total

1 Claude's Character (45 min)

Read Anthropic's official documentation on Claude's character and values. Understand what makes Claude different - the emphasis on being helpful, harmless, and honest.

Claude's Character Character Research Paper

2 System Prompts Deep Dive (45 min)

Learn how system prompts work and how to craft them effectively. Understand the difference between system prompts and user messages.

  • System prompt structure and best practices
  • Setting tone, style, and constraints
  • Role-playing and persona creation
System Prompt Guide Prompt Engineering

3 Hands-On Practice (45 min)

Create 3 different system prompts for different use cases:

  • A technical documentation writer
  • A sales discovery call assistant
  • A code review expert

4 Constitutional AI (45 min)

Understand how Claude is trained differently - Constitutional AI and RLHF. This is Anthropic's key differentiator.

Constitutional AI Paper Claude's Constitution

Day Deliverable

Create a document with 3 polished system prompts. Test each in Claude.ai and note what works well vs needs refinement. Save these for your portfolio.

Day 2: Prompting is the Product

LEARN

Learning Objective

Master advanced prompting techniques. Learn chain-of-thought, few-shot learning, and structured outputs. Understand how to get consistent, high-quality results.

Learning Blocks ~3.5 hours total

1 Chain-of-Thought Prompting (45 min)

Learn how to make Claude "think out loud" for better reasoning on complex tasks.

  • When to use CoT vs direct answers
  • Structuring reasoning steps
  • Using XML tags for clarity
Chain of Thought Guide

2 Few-Shot Learning (45 min)

Learn to provide examples that shape Claude's output format and style.

  • Selecting effective examples
  • Example formatting best practices
  • When few-shot beats zero-shot
Multishot Prompting

3 Structured Outputs (45 min)

Get Claude to output JSON, XML, and other structured formats reliably.

  • JSON mode and when to use it
  • XML tags for complex structures
  • Validation and error handling
XML Tags Guide

4 Prompt Library Study (60 min)

Study Anthropic's official prompt library. Analyze what makes these prompts effective.

Anthropic Prompt Library

Day Deliverable

Create a "Prompt Playbook" with 5 reusable prompts for common sales tasks: discovery call prep, objection handling, email drafting, competitive analysis, and account research.

2 Basic Workflows (60 min)

Learn the core Claude Code workflows:

  • Code generation and editing
  • File management and navigation
  • Running commands and tests

  • Planning and executing multi-file changes
